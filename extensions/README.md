`extensions`（插件）目录一般是在`C:\Users\Administrator\.vscode\extensions`目录下，如果有用户名，则需将`Administrator`改成你的`windows`系统用户名。

`Macbook`的则在`/Users/你的用户名/.vscode/extensions`目录下。
